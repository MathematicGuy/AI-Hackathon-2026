# Product Architecture

This directory contains product-specific system architecture. It is separate
from [`docs/ARCHITECTURE.md`](../../ARCHITECTURE.md), which defines generic
Harness discovery and boundary rules.

- [`air-conditioner-advisor-m1.md`](air-conditioner-advisor-m1.md) describes
  the Milestone 1 advisor architecture.

Product architecture must conform to the accepted product contract, Accepted
ADRs, and approved requirements. It cannot silently supersede them.
